📖 Publish Your Words

The Sync Bell converts your plain .txt files into beautiful HTML books instantly. Works 100% offline — no account or internet needed.

📋 How Does the Library Know What Books to Show?

The Sync Bell automatically creates a manifest.json file inside every library.zip it generates. This small index file lists all your books. Your library.html page reads it to display your full searchable book list automatically.

You never touch the manifest yourself. The flow is: write .txt file → drop on Sync Bell → download library.zip → extract → upload the library/ folder (which already contains manifest.json) → done.

To add a new book later: write new .txt file → drop all your books (old and new together) onto Sync Bell → download new zip → re-upload library/ folder. The manifest regenerates fresh every time. ✨
